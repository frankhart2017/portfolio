SRM Brain

Project Details

  • GitHub : SRM Brain
  • Technologies : HTML, CSS, JavaScript, jQuery, Python, Flask, Nginx
  • Category : Personal

A webapp that provides production ready APIs for several Deep Learning (and Machine Learning) tasks along with datasets and tools to clean data. The tool allows developers and AI researchers to try out their ideas in a short span of time without having to train computationally expensive models on their own. Another use case of this project is when developers who are not aware of deep learning (or AI in general) need to deploy a module which requires an AI model, these developers can simply download the production ready API and host it.
